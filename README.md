# Cwitter-V1.0
This is a mini Twitter named Cwitter.

# Software required
    JAVA 8 
    Mavan 3 

# How to Run
Fire up the command line; Navigate to the directory: "Cwitter-V1.0/" and run the following command:

    mvn compile exec:java
